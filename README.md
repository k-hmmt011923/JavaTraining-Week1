実行方法

1.ターミナルで src フォルダに移動　
cd src

2.Java ファイルをコンパイル
javac Janken.java

3.クラスファイルを実行
java Janken

工夫した点・学んだこと

全体の構成と必要な知識を細分化して整理し、参考書や生成 AI を活用しながらも、理解できた処理だけを取り入れて実装するよう心がけました。Scanner を使った入力バリデーションや、勝敗判定を (userHand - computerHand + 3) % 3 で簡潔に表現する方法を通して、基本構文を組み合わせて処理を整理する考え方を学びました。また、まず全体像を分解し「なぜ動くのか／なぜ動かないのか」を確認しながら進めることで、自分の理解不足に気づき、補いながら実装を進める大切さを実感しました。これらを通して、Java の基本的な流れと構造を体験的に理解することができました。
